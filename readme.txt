# 🌐 My Portfolio Website

A simple and responsive portfolio website built using **HTML, CSS, and JavaScript**.  
This project showcases my skills, projects, and contact information.  

---

## 📂 Project Structure

```
my_portfolio/
│── index.html   # Website structure (HTML)
│── style.css    # Website design (CSS)
│── script.js    # Website interactivity (JavaScript)
```

---

## 🚀 How to Run the Project

### Step 1: Clone the Repository
```bash
git clone https://github.com/your-username/my_portfolio.git
cd my_portfolio
```

### Step 2: Open in Browser
- Simply double-click **index.html**  
- Or open it inside your editor (Cursor / VS Code) and use **Live Server**  

---

## 🛠️ Technologies Used
- **HTML5** → Structure of the site  
- **CSS3** → Styling & responsive design  
- **JavaScript** → Basic interactivity

## ✨ Features
✔️ Responsive navigation bar  
✔️ About section  
✔️ Project showcase  
✔️ Contact details with email & LinkedIn  
✔️ Simple, clean, and beginner-friendly design  

---

## 🌍 Hosting with GitHub Pages

1. Go to **Settings** in your GitHub repo.  
2. Scroll to **Pages**.  
3. Under **Source**, select branch → `main` → Save.  
4. Your site will be live at:  
   ```
   https://your-username.github.io/my_portfolio/
   ```
